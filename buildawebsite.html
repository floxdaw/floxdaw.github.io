How I made this Website 

Mobirise + Github Pages

Originally, I tried to test and deploy a website using Wix's new AI feature, however that never loaded and seemed to not work. 

So I returned to what I know, as I have used Mobirise before. This was a good refresher for my GitHub knowledge too.

I created a Mobirise project, and saved that to a new folder on my PC called whatever you want. 
Then I crated the GitHub repo with the correct name based on this guide and linked it to the directory I have my Mobirise project on. Now when I publish the Mobirise project to folder, I can then push to git whenever I feel like.

It's also worth noting that I used Stable Diffusion XL to generate some of the images scattered around the website. I used Fooocus AI via Google Colab, which is completely free, and can be found here. I think it adds a cool aesthetic.
